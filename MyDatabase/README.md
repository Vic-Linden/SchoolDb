# Fictional School Database

This project is a console-based school management system built with **C#** and **.NET**, using **Entity Framework Core**, **SQL Server**, and **Spectre.Console** (for console UI).

The application is designed to handle students, staff, classes, courses, grades, and departments for a fictional school.

## Features

### Console Application (EF Core)
- View all students with their class, courses, grades, and teachers.
- View active courses.
- Count teachers per department.
- Add and delete students and staff.
- Set grades using transactions for data safety.
- Error handling.

### SQL (SSMS)
- Create and manage database structure.
- Insert and retrieve staff, students, grades, and courses.
- Salary summaries per department.
- Stored Procedure to fetch detailed student information by ID.
